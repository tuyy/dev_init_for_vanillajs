### vanilla js 기본 개발 환경 ``` $ git clone .. $ npm init $ npm install ### jest 테스트 실행 $ npm run test ### dev 환경에서 실행 $ npm run start ### static/main.js 생성 $ npm run build ``` #### test 작성 - src 하위에 app.spec.js 파일에 TC 작성 - 또는 {test name}.{test or spec}.js 파일 생성 후 TC 작성 #### npm run build - 최종 빌드된 파일은 ```static/main.js``` 에 생성됨
